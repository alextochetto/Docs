UpdateSector

The UpdateSector can be used to update a sector or switch the container of a sector

Parameters:

0 - SectorName - The sector that will be updated

1 - Url - The url that will be used to replace the sector

2 - Title - The title to be used

3 - CanRoute - Can the router be used

4 - CanLoadDefaultSectors - Can the default sector be loaded

5 - ContainerCode - The container code to be used to store the data when we switch the secto container

Example

UpdateSector(content,~/MyPage.html)

In this case the sector 'content' will be updated with the content of the page MyPage.html
