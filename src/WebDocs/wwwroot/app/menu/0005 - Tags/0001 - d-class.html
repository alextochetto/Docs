d-class

You can use d-class when you need to add conditional class to your element.

Syntax:

            d-class="{classToBeUsed:Conditional:classToBeUsedIfConditionalFails}"

The "Conditional" is optional. In this case it will be true. The "classToBeUsedIfConditionalFails" is optional too

You can use it with multiple classes too

Syntax:

            d-class="{classToBeUsed1:Conditional1:classToBeUsedIfConditionalFails1,classToBeUsed2:Conditional2:classToBeUsedIfConditionalFails2}"
