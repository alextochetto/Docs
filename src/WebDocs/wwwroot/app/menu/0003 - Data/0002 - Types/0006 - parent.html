parent

The data type parent can be used to read an element in the hierarchy of the page. The attribute d-dataValue will be used to search the first parent with attributes begining with its value. If the d-dataValue is not present, the first parent will be used.

Simple Example:

            <div class="state" dc-levels="0" dc-children="1"> 
                <div d-dataKey="internal" d-dataType="parent" d-dataValue="dc-"></div>
            </div>

In this case the storage item internal will be created as an object with 2 properties inside it. That is levels with value of 0 and children with the value 1


You can use d-dataLoadType equals to reference when you need that all mustaches must be evaluated recursively.

Recursive Example:

            <div d-dataKey="source" d-dataType="object" d-dataProperty-levels="2" d-dataProperty-children="5"></div> 
            <div class="state" dc-levels="{{source.levels}}" dc-children="{{source.children}}"> 
                <div d-dataKey="internal" d-dataType="parent" d-dataLoadType="reference" d-dataValue="dc-"></div> 
            </div>

In this case the storage item internal will be created as an object with 2 properties inside it. That is levels with value of 2 and children with the value 5
